
Project Overview: Social.az (Instagram Clone)
Social.az is a high-performance, minimalist social media application inspired by the modern Instagram UI. Built with a focus on speed and responsiveness, the project transitions from traditional component libraries to a custom, utility-first styling approach to ensure a seamless user experience.

🚀 Tech Stack
Frontend Framework: React.js (Vite-powered)

Styling: Tailwind CSS (Optimized for performance)

State Management: React Hooks (useState, useMemo, useCallback)

Environment: CachyOS (Linux) optimized build workflow

✨ Key Features
Optimized Feed System: A lightning-fast post rendering engine using React.memo to prevent unnecessary re-renders.

Modern Instagram UI: A clean, centered layout with a sticky navigation bar, a left-side menu, and a dynamic "Suggested for You" right sidebar.

Dynamic Post Creation: A custom-built, lightweight modal system for sharing thoughts and media without compromising system resources.

Responsive Design: Fully adaptive layout that shifts from a three-column desktop view to a focused single-column mobile view.

Efficient Asset Loading: Implements lazy-loading for images and optimized icon imports to ensure low CPU and RAM usage.

🛠 Technical Highlights
Performance Optimization: Resolved "Main Thread" bottlenecks by migrating from heavy Material UI components to streamlined Tailwind CSS utilities.

Memory Management: Fine-tuned for Linux environments, ensuring smooth operation even during intensive development tasks (Vite HMR).

Modular Architecture: Components are strictly decoupled, making the codebase easy to maintain and scale.

📸 UI Components
Navbar: Features a global search bar and quick-access action icons.

Sidebar: A minimalist navigation tool for exploring home, profile, and settings.

Feed: The heart of the app, displaying user content in a sleek, borderless card format.

Rightbar: Enhances user engagement by showcasing online friends and suggested photos.

Developer Note: This project demonstrates the transition from "heavy-weight" component libraries to "utility-first" CSS, proving that high-quality UI/UX can coexist with peak performance.

How to Run
Clone the repository.

Install dependencies: npm install.

Run the development server: npm run dev.
